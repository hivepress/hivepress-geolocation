Geolocation extension for HivePress plugin.

== Description ==

HivePress Geolocation is an extension for [HivePress](https://wordpress.org/plugins/hivepress/) plugin. It allows users to search listings by location.

[Demo](https://demo.hivepress.io/) | [Docs](https://hivepress.io/docs/extensions/geolocation/) | [Support](https://hivepress.io/support/forum/extensions/geolocation/)
