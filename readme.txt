== Description ==

HivePress Geolocation is an extension for [HivePress](https://wordpress.org/plugins/hivepress/) plugin. It allows users to search listings by location.

[Demo](https://listinghive.hivepress.io/) | [Docs](https://help.hivepress.io/category/39-geolocation) | [Support](https://hivepress.io/support/forum/extensions/geolocation/?utm_medium=referral&utm_source=wordpress.org)
